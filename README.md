# サイト概要

MacroStock：ここに URL

MacroStock(マクロストック)はマクロ写真を専門としたフォトストックサイトです。

マクロ写真とは、植物や虫などの小さい物をカメラの機能を使って一枚の写真に大きく写し出す写真のことです。

普段、ブライダル写真を中心にプロのフォトグラファーとして仕事をしています。

あまり撮影することのないマクロ写真という領域の作品を残したいと思いこのサイトを作成しました。
